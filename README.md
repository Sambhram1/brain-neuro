# TRIBE v2 — Neural Response Analyzer

Upload a video and predict cortical brain activity using Meta's TRIBE v2 model.

## Setup

This Space requires a **HF_TOKEN** secret with access to:
- [meta-llama/Llama-3.2-3B](https://huggingface.co/meta-llama/Llama-3.2-3B) (gated model — request access first)

Add it in **Settings > Secrets** as `HF_TOKEN`.
